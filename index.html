STEP 1. 카테고리 선택

  • 대분류(의류/악세) → 소분류 순서로 선택
  • 카테고리 미선택 시 이미지 분석 불가

STEP 2. 이미지 업로드

  • 슬롯 1: 만들고 싶은 컷의 원본 이미지
    (정면컷 → 정면 원본 / 후면컷 → 후면 원본)
  • 슬롯 2~5: 디테일 이미지 + 설명란 작성

슬롯 설명란 작성 규칙

  • 위치 포함 필수 — "하단 우측 금속 로고"
  • 위치 없으면 X — "로고 디테일"
  • 측면에만 있는 디테일 → "정면에서 보이지 않음, 소재/하드웨어 참고용"
  • 구현 안 할 디테일 → "구현하지 않을 것" 명시
  • 소재 특성 힌트 → "소재감 참고 (시어함, 시스루)" 형식으로 작성

STEP 4. 이미지 분석

  • 분석 후 소재 / 컬러 / 디테일 결과 반드시 확인
  • 잘못 분석된 내용은 직접 수정
  • 같은 제품 여러 컷 작업 시 🔒 잠금 기능 활용
    (소재/컬러 잠금 → 재분석해도 덮어쓰기 안 됨)
  • 상충 표현 주의 — "크리스피 바스락" + "부드러운" → AI가 부드럽게 타협함

주의사항

  • 디테일 슬롯에 정면 전용 디테일 보이면 설명란에 "구현하지 않을 것" 명시
  • 후면에 없는 정면 디테일은 최종 디테일란에서 직접 삭제
  • 소재/컬러는 정면 분석 기준으로 🔒 잠금 후 작업
  • 히스토리 기능으로 정면 분석 내용 불러오기 가능

클로즈업컷 제작 시 주의사항

  • 정면/후면컷을 크롭해서 쓸 수 있으면 클로즈업 생략 가능
  • 클로즈업이 필요한 경우에만 별도 제작

작업 순서

1)카테고리 선택 → 컷 종류: 클로즈업 → 부위 선택
2)슬롯 1에 해당 부위 원본 이미지 업로드
3)이미지 분석 실행
4)프롬프트 생성

같은 제품 여러 클로즈업컷 제작 시

  • 컷마다 슬롯 1 이미지 교체 후 이미지 분석 다시 실행
  • 부위 퀵셀렉터 변경 후 프롬프트 재생성
  • 예외: 원단컷은 이미지 분석 없이 프롬프트 생성 가능
